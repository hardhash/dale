# Matplotlib常见的一些注意事项和API
****
## 常见操作

- 科研绘图一般不需要设置图表长宽，往往出版社对图表分辨率有明确要求，一般来说是不低于300dpi，具体可查阅期刊相关文件，对于dpi不低于300可直接设置`plt.figure(dpi=300)`
